You are now ready to start developing using Proxima.

This starter template, doesn't include any third party library but those needed for our custom design:

  • ngx-bootstrap
  • ngx-perfect-scrollbar
  • ng2-file-upload
  • angular-feather

You're free to install and use any third party library of your preference.

What's next?

You might want to start by defining your own custom routes and enabling the navigation for those.

Enabling navigation

The navigation menu vertical and horizontal is done automagically through a NavigationService, you just need to configure the json structure and you're done.

Please open app/layout/services/navigation.service.ts file and configure your navigation options. Refer to the full template to see how it can be nested.
